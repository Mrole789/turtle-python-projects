# turtle-python-projects
Collection of my python turtle graphic scripts.

## Projects include

**Ghana flag**: Draws the flag of ghana with accurate colors and star  
**Hirst Spot Paint**: Creates a Damien Hirst-Inspired spot painting  
**Random Walk**: A colorful random walk with random changing directions  
**Turtle Race**: Animated race with multiple turtle instances  
**Sketch pad**: A simple interactive drawing application

Controls for sketch pad  
   'w': Move forward  
   'z': Move backward  
   'a': Turn right  
   's': Turn left  
   'c': Clear screen  
   'u': Pen up  
   'd': Pen down


##Requirements

-Python 3.x
-Python turtle module (built-in)
